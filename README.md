# Free Look Mod for Minecraft 1.21.11 (Fabric)

## Installation

1. Make sure you have **Minecraft 1.21.11** and **Fabric Loader** installed.
2. Download the mod `FreeLook 1.21.11fb.jar` file.
3. Place the `FreeLook 1.21.11fb.jar.jar` file into your `mods` folder (located in your Minecraft game directory). Create the `mods` folder if it doesn't exist.
4. Launch the game using the **Fabric** profile.
5. The mod is now active. Enjoy!

## Changing the Keybind (Optional)

To change the default Free Look key:

- Go to `Settings` → `Options` → `Controls` → `Key Binds`
- Find the **Miscellaneous** category
- Look for **rqu Free Look**
- Click on it and press your desired key

Good luck! :P
